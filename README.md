# 쿠팡 로켓그로스 매출·이익 계산기

쿠팡 로켓그로스에서 판매된 상품의 **매출·광고비·마진(이익금)** 을 상품별·일자별로 자동 산출하는 웹 도구입니다.
브라우저에서만 동작하며 **업로드한 파일과 계산 결과는 외부로 전송되지 않습니다.**

## 사용 링크
👉 https://wmh1295-sudo.github.io/coupang-margin

## 사용 방법
1. **마진표** 엑셀을 한 번 올립니다. (등록상품ID별 `결제받는단가`·`마진`이 적힌 파일 → 브라우저에 기억됨)
2. 매일 **비즈니스 인사이트 엑셀**(`SELLER_INSIGHTS_VENDOR_ITEM_METRICS...`)과 **광고 리포트 엑셀**(`..._YYYYMMDD_YYYYMMDD.xlsx`)을 드래그합니다. (종류 자동 인식)
3. **계산하기** → 상품별 표 확인 → **엑셀 다운로드**.
4. 다른 날짜 파일을 넣으면 결과가 **누적**되어 일자별로 쌓입니다.

## 계산 방식 (등록상품ID 기준)
| 항목 | 계산 |
|---|---|
| 판매수량 | Σ 인사이트 판매량 (옵션ID → 등록상품ID 합산) |
| 인사이트매출 | Σ 인사이트 매출(원) (판매가 기준) |
| 정산매출 | 결제받는단가 × 판매수량 |
| 사용광고비 | Σ 광고비 (광고 옵션ID → 등록상품ID 매핑 후 합산) |
| 마진(광고전) | 개당마진 × 판매수량 |
| 순이익 | 마진(광고전) − 사용광고비 |

- 광고 리포트에는 등록상품ID가 없어, 인사이트 파일의 **옵션ID→등록상품ID 매핑**을 거쳐 광고비를 상품에 붙입니다.
- 판매됐지만 마진표에 없는 상품, 매칭되지 않은 광고비는 **경고 패널**로 표시됩니다(조용히 누락시키지 않음).
