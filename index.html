쿠팡 로켓그로스 매출·이익 계산기

인사이트 · 광고 리포트 엑셀을 넣으면 상품별 매출·광고비·마진을 자동 산출합니다. 모든 계산은 이 브라우저 안에서만 이뤄지며 파일은 외부로 전송되지 않습니다.

1. 마진표

없음

등록상품ID별 결제받는단가·마진이 적힌 엑셀입니다. 한 번 올리면 이 브라우저에 기억되어, 이후에는 다시 올릴 필요가 없습니다.

2. 인사이트 + 광고 리포트

두 파일을 아래에 드래그하세요. 파일 종류(인사이트/광고)는 자동 인식됩니다. 날짜는 광고 리포트 파일명에서 읽습니다.
여러 날을 모으려면 매일 파일을 넣고 계산하면 자동으로 누적됩니다. (이전에 다운로드한 결과 엑셀을 여기 넣으면 이어서 기록)

📄
여기에 엑셀 파일을 드래그하거나
인사이트 · 광고 리포트 (한꺼번에 여러 개 가능)

3. 결과

☁️ 구글시트 저장 (선택)

미연결

연결하면 계산할 때마다 본인 구글시트에 자동 누적됩니다(일별요약·상세 탭). 설정 방법은 apps-script.gs 파일 상단 주석을 참고하세요.
